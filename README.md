## 特性

一次全局安装，所有的开发项目都支持，不需要每个项目都安装配置webpack。

无需熟悉webpack，开箱即食。

支持es2015。

支持热替换。

支持反向代理。

默认支持vue，支持react。

支持各种css预处理（less，sass，postcss。。。），默认支持less。

zls.config.js配置文件即可使用。

## Install

`npm install -g zls-cli`

## Use

```
zls init app

cd app

npm run dev
```
